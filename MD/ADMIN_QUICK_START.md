# راهنمای سریع دسترسی به پنل مدیریت

## دسترسی به پنل مدیریت

پس از ایجاد superuser، می‌توانید از طریق آدرس زیر به پنل مدیریت دسترسی پیدا کنید:

### آدرس پنل مدیریت:
```
http://127.0.0.1:8000/admin/
```

یا اگر آدرس ادمین را تغییر داده‌اید:

```
http://127.0.0.1:8000/{ADMIN_URL}/
```

## مراحل دسترسی

### 1. راه‌اندازی سرور (اگر در حال اجرا نیست)

```bash
# فعال‌سازی محیط مجازی
source venv/bin/activate

# راه‌اندازی سرور
python manage.py runserver
```

### 2. باز کردن مرورگر

مرورگر خود را باز کنید و به آدرس زیر بروید:
```
http://127.0.0.1:8000/admin/
```

### 3. وارد کردن اطلاعات ورود

از اطلاعات superuser که ایجاد کرده‌اید استفاده کنید:
- **Username**: نام کاربری superuser
- **Password**: رمز عبور superuser

### 4. دسترسی به پنل

پس از ورود موفق، به پنل مدیریت Parsa Journal دسترسی خواهید داشت.

## ویژگی‌های پنل مدیریت

پنل مدیریت شامل بخش‌های زیر است:

- **Users**: مدیریت کاربران
- **Authors**: مدیریت نویسندگان
- **Articles**: مدیریت مقالات
- **Categories**: مدیریت دسته‌بندی‌ها
- **Tags**: مدیریت تگ‌ها
- **Book Reviews**: مدیریت نقدهای کتاب
- **Movie Reviews**: مدیریت نقدهای فیلم
- **Newsletter**: مدیریت خبرنامه
- **Comments**: مدیریت نظرات
- **Site Settings**: تنظیمات سایت
- **Contact Messages**: پیام‌های تماس

## تغییر آدرس ادمین (برای امنیت بیشتر)

برای تغییر آدرس ادمین از `/admin/` به آدرس دلخواه:

1. فایل `.env` را باز کنید
2. متغیر `ADMIN_URL` را تغییر دهید:

```bash
ADMIN_URL=my-secret-admin-panel
```

3. سرور را دوباره راه‌اندازی کنید

حالا پنل مدیریت در آدرس زیر در دسترس خواهد بود:
```
http://127.0.0.1:8000/my-secret-admin-panel/
```

## امنیت

### پنل دیفالت Django غیرفعال است

پنل دیفالت Django (`/admin/` اگر `ADMIN_URL` تغییر کرده باشد) کاملاً غیرفعال و غیرقابل دسترسی است. فقط پنل مدیریت اختصاصی Parsa Journal در دسترس است.

### ویژگی‌های امنیتی

1. **محدودیت دسترسی**: فقط کاربران با `is_staff=True` می‌توانند وارد شوند
2. **Rate Limiting**: حداکثر 5 تلاش برای ورود در هر 15 دقیقه
3. **لاگ‌گیری**: تمام تلاش‌های ورود و دسترسی در فایل‌های لاگ ثبت می‌شوند
4. **محدودیت IP (اختیاری)**: می‌توانید دسترسی را به IP های خاص محدود کنید

## عیب‌یابی

### نمی‌توانم وارد شوم

1. **بررسی کنید که کاربر superuser است:**
   ```bash
   python manage.py shell
   ```
   ```python
   from accounts.models import User
   user = User.objects.get(username='your_username')
   print(f"is_superuser: {user.is_superuser}")
   print(f"is_staff: {user.is_staff}")
   print(f"is_active: {user.is_active}")
   ```

2. **اگر کاربر superuser نیست، آن را تبدیل کنید:**
   ```bash
   python manage.py create_admin --username your_username
   ```

### خطای "Too many login attempts"

- 15 دقیقه صبر کنید
- یا کش را پاک کنید (فقط در محیط توسعه)

### آدرس پنل کار نمی‌کند

1. بررسی کنید که `ADMIN_URL` در `.env` صحیح است
2. سرور را دوباره راه‌اندازی کنید
3. بررسی کنید که URL در مرورگر صحیح است
